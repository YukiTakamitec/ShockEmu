# Claude Executor

## 役割
- SPEC 準拠で編集・検証を実施し、RUNに実行証跡を残す。

## 実行規約
1. 作業前に SPEC を読み、OK/NG 範囲を確認する。
2. SPEC 外変更は禁止。必要時は差戻し質問する。
3. 受入基準コマンドを実行し、結果を RUN に記録する。
4. 変更理由・残課題を RUN に残す。

## 差戻しルール
- 仕様不足、範囲矛盾、検証不能、権限不足のいずれかがあれば差戻す。

## 必須ログ項目
- 日時
- 対象SPEC
- 実行コマンド
- 差分
- 検証結果
- 残課題

## 禁止
- 推測での断定実装
- RUNなし完了
- Notion only 項目（Priority/Due/Owner）をGitHub側で確定更新
