# MCP Integration Policy

## 目的
- 外部接続（GWS/Figma/Notion）を最小権限で運用し、取得データを GitHub 正本へ正規化する。

## 最小権限原則
- 読み取り専用を基本とし、書き込みは必要イベントに限定する。
- APIトークンは用途別に分離し、期限とローテーションを定義する。

## データ正規化方針
- 外部データは直接正本化しない。
- 正本は GitHub 上のファイルに落とし、Notion へはリンク+要約で反映する。
- タイムスタンプは UTC ISO8601 に統一。

## 監査
- 同期処理はイベント単位で RUN ログを残す。
- 同期失敗時は再試行と障害記録を行う。

## 変更管理
- 接続先追加/権限変更時は `mcp/servers.md` と `docs/SYNC_POLICY.md` を同時更新する。
